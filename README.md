## หลังจากติดตั้งแล้วให้เพิ่มคำสั่งต่อนี้ไป
- กด Ctrl + Shift + P 
- กรอกคำว่า preferences:Open Settings (JSON) ลงไป
- คัดลอกคำสั่งด้านล่างไปใส่แล้ว เปิดปิดโปรแกรมใหม่
```
    },

    "editor.snippetSuggestions":"top",
    "editor.tabCompletion":true
       
}
```
